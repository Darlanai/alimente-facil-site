Política de Privacidade

Esta política explica como os dados do usuário são tratados no Alimente Fácil.

1. Dados coletados

Podem ser coletados nome, e-mail, informações de autenticação, status de assinatura e dados necessários para o funcionamento do sistema.

2. Finalidade

Os dados são usados para autenticação, controle de acesso, comunicação do usuário, suporte e operação da assinatura.

3. Pagamentos

Os pagamentos recorrentes são processados pelo Mercado Pago. Dados sensíveis de cartão não são armazenados diretamente pelo Alimente Fácil.

4. Compartilhamento

Os dados só são compartilhados com provedores estritamente necessários à operação, como hospedagem, banco de dados, autenticação e pagamento.

5. Segurança

São adotadas medidas razoáveis de segurança, mas nenhum sistema online é absolutamente invulnerável.

6. Direitos do usuário

O usuário poderá solicitar atualização ou exclusão de dados, observadas as obrigações legais e operacionais aplicáveis.

7. Contato

Para qualquer solicitação relacionada à privacidade, escreva para projetosdarlan@gmail.com.
